1. uname -a
Інформація про поточну ОС (ім'я, реліз та версія ядра, апаратна платформа, архітектура, тип процесора)

2. sudo apt upgrade
Оновлення кожного застарілого пакета та залежностей у системі

3. cat /etc/apt/sources.list
Список репозиторіїв, звідки були інстальовані пакети та програми

4. sudo apt-get install vim vim-gtk 
Встановлення текстового редактора Vim GTK

5. cd path_to_dir
Зміна поточний каталогу на вказаний як параметр

6. ls, dir
Вивід списку файлів у каталозі

7. ls -la
Можливість бачити приховані файли + права та розмір

8. dpkg -i <ім'я_файлу_пакету> (error: requested operation requires superuser privilege)
Потрібно залогінитись під рутом. Встановлює цей пакет

9. sudo su
Авторизація від імені адміністратора

10. echo 'something'
Виведення тексту в консоль

11. touch <ім'я_файлу>
Створення файлу у поточній директорії

12. vi - не знаючи 2 режими роботи з редактором, не вийде нічого написати (введення - i, командний режим - Esc)
:q! - вихід, :w - зберегти, :wq - зберегти та вийти

13. Скрипт (ім'я script.sh)
Перший виклик ./script.sh не дозволить виконання, тому що немає прав на файл
-------------------
#! /bin/sh
echo "something $1"
-------------------

14. chmod [<опції>] <права> <шлях_до_файлу>
r - читання, w - запис, x - виконання, s - від прав суперкористувача
Категорії користувачів: u - власник, g - група, o - інші користувачі
Вісімковий формат:
0 - жодних прав
1 - виконання
2 - запис
3 - виконання та запис
4 - читання
5 - читання та виконання
6 - читання та запис
7 - усі права

15. cp ./script.sh <ім'я_директорії>
cp -r <директорія_1> <директорія_2> - рекурсивне копіювання з папки до папки

16. mv ./script.sh <ім'я_директорії>
Переміщення файлу

17. rm <ім'я_файлу>
Видалення файлу

18. rm -rf <ім'я_директорії>
Видалення директорії

19. tar cvf <ім'я_архіву> <ім'я_файлу>
Копіює файл або папку з файлами в стислий архів

20. tail -n <ім'я_рядок> <файл>
Читає останні n рядків у файлі

21. find. -name *dhcp*
Пошук файлів за регулярним виразом

22. ps aux
Список усіх процесів

23. kill <PID>
Перериває роботу процесу

24. dmesg
Вивід повідомлення ядра при початковому завантаженні Linux або налаштовує їх буфер

25. ipconfig
Інструмент для роботи з мережею

26. diff <файл_1> <файл_2>
Пошук відмінностей між двома файлами

27. df
Звіт про використання дискового простору
